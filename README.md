# Portfolio

A premium personal portfolio website built with the MERN stack. Features a modern dark theme with responsive design, interactive components, and a contact form with email notifications.

## Tech Stack

- **Frontend:** React, Vite, Tailwind CSS, Framer Motion
- **Backend:** Node.js, Express.js, MongoDB, Nodemailer
- **Design:** Dark theme (#0F172A) with Primary #6366F1

## Sections

- Hero with typewriter effect
- About with highlights
- Skills with animated progress bars
- Projects with filterable cards
- Experience timeline
- Testimonials carousel
- Contact form with email notifications

## Getting Started

```bash
npm install
cd client && npm install
cd ../server && npm install
cd ..
npm run dev
```
